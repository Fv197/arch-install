# arch install

Personal installation script for Arch Linux on Thinkpad T560 with archlinux-2023.01.01-x86_64

## Description

* Base installation
* BTRFS
  - Snapper
  - Swapfile
* Hibernation
* GRUB

## Instructions

1. Clone this repo
2. Edit `arch-install.sh` (variables in start of file)
2. Run `./arch-install.sh`
